### 🤖 AI-Based Software Learning Assistant
> An interactive AI-powered assistant for software learning, built using FastAPI for the backend, LLMs for intelligent responses, and RAG for grounded, context-aware answers — delivered through a Streamlit interface.

| Layer | Technology |
|---|---|
| Backend | FastAPI |
| Intelligence | LLMs, Retrieval-Augmented Generation (RAG) |

### 📚 AI-Powered Research Paper Summarizer & Knowledge Retrieval System
> A semantic search and summarization system that helps retrieve and condense research paper knowledge, built with LangChain for orchestration, Ollama for local LLM inference, and FAISS for fast vector-based semantic search.

| Layer | Technology |
|---|---|
| Orchestration | LangChain |
| LLM Runtime | Ollama |
| Vector Search | FAISS |
